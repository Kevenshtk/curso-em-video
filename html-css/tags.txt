Formatações 
<strong> deixa em negrito 

<q> usado para citações simples("")

<blockquote> usado para citações completas(tem quebra de linha e um espaço)

<small> deixa o texto pequeno

<sub> H2O <p>H<sub>2</sub>O</p>

<abbr> coloca o significado das abreviações

<hr> criar uma divisão entre dois elementos

ESTRUTURA
<header> é o cabeçalho do site
<main> é o assuto  principal dentro do <body>
<section> é uma seção do site
<footer> é o rodapé do site
<aside> conteúdo extra 

TABELAS
<table>
<colgroup> <col> usado para seleciona colunas 
<caption> é a lengenda da tabela
<tr> linha da tabela
<td> dado da tabela, colsan="x" mescla células em coluna, rowsan="x" mescla células em linha
<thead> cabeçalho da tabela
<th> título na tabela, scope="col(coluna), colgroup(grupo), row(linha), rowgroup(grupo)" diz de onde são os dados de th 
<tbody> corpo da tabela
<tfoot> rodapé da tabela

IFRAME
<iframe> faz acessa um outro site sem sair do site atual O.B.S.: no meio do iframe pode coloca avisos, 
com o parâmetro (name) posso fazer o iframe de tela para links colocando no (target)
srcdoc permite usar html dentro do iframe

FORMULÁRIOS
<label> faz o campo se relaciona com o elemento

<input>
required faz o campo ser obrigatório
minlength="5" faz o campo ter um minímo de caracter
maxlength="8" faz o campo ter um máximo de caracter
size="8" tamanho do campo
placeholder="Nome do usuário" mostra uma dica dentro do campo 
autocomplete="username" indica o tipo do campo
value="F" faz ter um valor quando escolhido

INPUT NUMBER
min="0" valor minímo aceito
max="10" valor máximo aceito
step="0.5" faz o valor muda de 0.5

input:checkbox pode escolher vários
input:radio só pode escolher um (todos do mesmo grupo precisa ter o mesmo name)
input:range traz um botão para arrasta (value="" indica o valor inícial) 

INPUT SELECT
selected deixa o item já selecionado 
option são os itens da lista
<optgroup label="nome do grupo">

OUTPUT
<output name="" id="">0</output> ele só monstra informações na tela e não envia
