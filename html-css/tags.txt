META TAGS
 <meta name="robots" content="noindex, nofollow(não nos associa aos links da página)" /> impede a página de aparecer em resultados de busca
 <meta name="description" content="descrição"> faz uma descrição para o navegador

Formatações 

<em> efatiza um texto

<mark> destaca um texto

<del> risca o texto

<q> usado para citações simples("")

<blockquote> usado para citações completas(tem quebra de linha e um espaço)

<small> deixa o texto pequeno

<sup> X^20 <p>x<sup>20</sup></p>

<sub> H2O <p>H<sub>2</sub>O</p>

<abbr> coloca o significado das abreviações

<hr> criar uma divisão entre dois elementos

<pre> O texto é mostrado como no código

<code> mostra um pedaço de código no site

ESTRUTURA
<header> é o cabeçalho do site

<main> é o assuto  principal dentro do <body>

<section> é uma seção do site

<footer> é o rodapé do site

<aside> conteúdo extra 

<article> deve fazer sentido por si só e deve ser possível distribuí-lo independentemente do resto do site.

<figure> é usada  para a inserção de uma figura
<figcaption> para incluir a descrição dessa figura

<em> é utilizado quando desejamos enfatizar um trecho ou palavra no texto

<cite> é utilizado para declarar que naquele trecho há uma citação
<q>responsável por apresentar o conteúdo retirado de outra fonte.
ex.:<q>Lorem ipsum dolor sit amet, consectetur </q> - <cite>http://br.lipsum.com/</cite>

<time> é utilizado para representar datas
ex.:<time datetime=”2017-04-07”>4/7</time>



TABELAS
<table>
<colgroup> <col> usado para seleciona colunas 
<caption> da um título para a tabela
<tr> linha da tabela
<td> dado da tabela, colsan="x" mescla células em coluna, rowsan="x" mescla células em linha
<th> cria uma coluna de cabeçalho, scope="col(coluna), colgroup(grupo), row(linha), rowgroup(grupo)" diz de onde são os dados de th 
<thead> cabeçalho da tabela
<tbody> corpo da tabela
<tfoot> rodapé da tabela



IFRAME
<iframe> faz acessa um outro site sem sair do site atual O.B.S.: no meio do iframe pode coloca avisos, 
com o parâmetro (name) posso fazer o iframe de tela para links colocando no (target)
scrolling="auto, (no)" esconde a barra de rolagem
srcdoc permite usar html dentro do iframe


FORMULÁRIOS
<form>
action é a url para onde os dados do formulário serão enviados.

method="get(mostra os dados enviados), post(não mostra os dados enviados)"

<textarea> é um campo para textos grandes

<label> faz o campo se relaciona com o elemento

<input>
required faz o campo ser obrigatório
minlength="5" faz o campo ter um minímo de caracter
maxlength="8" faz o campo ter um máximo de caracter
size="8" tamanho do campo
placeholder="Nome do usuário" mostra uma dica dentro do campo 
autocomplete="username" indica o tipo do campo
value="F" faz ter um valor quando escolhido

INPUT NUMBER
min="0" valor minímo aceito serve para input date ex.: min="2005-01-01"
max="10" valor máximo aceito
step="0.5" faz o valor muda de 0.5

INPUT CHECKBOX pode escolher vários
para permitir escolher mais de um checkbox é necessário eles estarem com o mesmo nome e no final [].

é usado o <label> para identificar a opição. ex.: <label>Futebol</label>

para marcar o checkbox cliclando no label é preciso adicionar um id no checkbox e um for com o mesmo no do id no label.

INPUT RADIO só pode escolher um (todos do mesmo grupo precisa ter o mesmo name)
o atributo cheched faz a opção ser marcada por padrão.
também aceita o required.

input:range traz um botão para arrasta (value="" indica o valor inícial) 

INPUT SELECT
<select> usado para criar uma lista.
selected deixa o item já selecionado.
multiple deixa o usuário escolher mais de uma opção (É necessário ter [] no final do nome).
size="quant" exibe a quantidade de opções indicada;

<option> conteúdo da lista.
<optgroup label="nome do grupo"> cria um grupo de opções dentro do select

FIELDSET
<fieldset> agrupa campos do form.
<legend> é o título do fieldset.

OUTPUT
<output name="" id="">0</output> ele só monstra informações na tela e não envia
